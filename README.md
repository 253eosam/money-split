# 💸 Money Split

**간단하고 직관적인 여행 정산 도우미**  
MoneySplit은 여행 중 사용한 지출 내역을 빠르게 정리하고, 친구들과 링크만으로 손쉽게 공유할 수 있는 경량 웹 서비스입니다.  
계정 없이도 사용 가능하며, 여행이 끝난 후 서로의 정산 내역을 확인하고 수정할 수 있어요!

## ✨ Features

- 📱 **모바일 최적화** UI
- 🔐 **로그인 없이 사용** 가능
- 🔗 **링크 공유를 통한 협업**
- 🧾 **카테고리별 소비 내역** 정리
- 🟢 **누가 정산했는지, 아직 미정산인지** 바로 확인

## 🛠 Tech Stack

- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Zustand** (상태 관리)
- **Vercel** (배포)

## 🚀 Getting Started

### 1. 프로젝트 클론

```bash
git clone https://github.com/253eosam/money-split.git
cd money-split
```

### 2. 의존성 설치

```bash
pnpm install
```

### 3. 로컬 개발 서버 실행

```bash
pnpm dev
```
